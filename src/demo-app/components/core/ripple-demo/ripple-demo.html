Ripple

MDC Ripple provides components (or any element) with a material "ink ripple" interaction effect.
Directive
mdc-ripple
Properties Description
boolean Activate or de-activate ripple ink effect. (Default is true)
Card - Click me 
<mdc-card [mdc-ripple]="true">
  <mdc-card-primary>
    <mdc-card-title>My card - Click me</mdc-card-title>
    <h2 mdc-card-subtitle>Subtitle</h2>
  </mdc-card-primary>
</mdc-card>
