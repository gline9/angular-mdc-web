Slider

Sliders let users select from a range of values by moving the slider thumb.
Component
mdc-slider
Properties Description
min: number The minimum value that the slider can have.
max: number The maximum value that the slider can have.
value: number The current value of the slider.
step: number Specifies the increments at which a slider value can be set.
discrete: boolean Discrete sliders allow users to select a specific value from a range. (Default: false)
markers: boolean Show markers on track. Discrete sliders support displaying markers on their tracks. (Default: false)
disabled: boolean Disables the toggle slider.
Methods Parameters
setMin(amount: number) Sets the min value the slider can have.
getMin(): number Returns the min value the slider can have.
setMax(amount: number) Sets the max value the slider can have.
getMax(): number Returns the max value the slider can have.
setStep(amount: number) Sets the slider step value to the given amount.
getStep(): number Returns the slider step value.
Events Description
change(source: MdcSlider, value) Broadcast when slider value is changed and committed by way of a user event, e.g. when a user stops dragging the slider or changes the value using the arrow keys.
input(source: MdcSlider, value) Broadcasts when slider value is changed by way of a user event, e.g. when a user is dragging the slider or changing the value using the arrow keys.
Continuous Slider 
<mdc-slider 
  [min]="0"
  [max]="100"
  [value]="0"
  [(ngModel)]="50"
  [disabled]="false"
  (input)="myInputEvent($event)"
  (change)="myChangeEvent($event)">
</mdc-slider>
