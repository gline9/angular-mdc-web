Floating Action Button

The MDC FAB component is a spec-aligned button component adhering to the Material Design button requirements.
Component
button[mdc-fab]
a[mdc-fab]
Properties Description
mini: boolean Make the fab smaller (40 x 40 pixels).
exited: boolean Animates the FAB in or out of view.
tabIndex: number Set the tabindex of the button. (Default: 0).
position: string Set absolute position. Valid values: 'bottom-left' | 'bottom-right' | null
Methods Description
toggleExited(exited?: boolean) Toggle the fab animation in and out. Optional argument for specifying exited. 
<button mdc-fab
  [mini]="false">
  <mdc-icon>edit</mdc-icon>
</button>
Exit/Entry Animation 
handleFabExitedClick()
  this.fab.toggleExited();

<button mdc-fab #fab
  (click)="handleFabExitedClick();">
  <mdc-icon>search</mdc-icon>
</button>
